# 미디어 세부 사항{#media-detail}

미디어 세부 사항 대시보드는 Concurrent Viewer 또는 Listener 수, 컨텐츠 시작, 완료율, 체류 시간 및 광고 시작을 포함하여 모든 컨텐츠에 대한 세부 지표를 표시합니다.
